# Browse category sidebar polish

Captured from the real local ClawHub app at `http://localhost:3000` using the
local public corpus.

- Desktop Skills and Plugins render their category lists in the left sidebar.
- Mobile Skills and Plugins keep categories in the top toolbar and hide the sidebar.
- Both mobile pages fit at `390x844` without horizontal overflow.
- No visible `Loading results` text remains after either page settles.
- The browse segmented control resolves to the shared `0px` design-system radius.
